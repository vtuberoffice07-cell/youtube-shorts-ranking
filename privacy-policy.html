1. 概要

「VTuber Buzz Ranking」（以下「本サービス」）は、個人VTuber（バーチャルYouTuber）の動画・投稿の動向を観察するための個人運営のオープンソースプロジェクトです。本サービスは YouTube Data API v3、TikTok 関連 API、Twitter(X) 関連 API を利用して公開情報を収集し、ランキング形式で表示します。

2. 収集する情報

2.1 本サービス利用者から収集する情報

本サービスは静的な HTML / JSON ファイルを GitHub Pages 上で配信しており、サーバーサイドで利用者の個人情報を収集することは 一切ありません。具体的に：

  • 利用者登録、ログイン、認証は 不要
  • Cookie やセッション ID を発行しません
  • 利用者の IP アドレス、デバイス情報、行動ログを取得しません
  • アクセス解析ツール（Google Analytics 等）を導入していません

2.2 外部 API から取得する情報

本サービスは以下の API から、各プラットフォームが 公開している情報 のみを取得します：

YouTube Data API v3 から取得する情報:
  • 動画ID、タイトル、説明文、サムネイル URL、公開日時、動画長
  • 動画の再生数、いいね数、コメント数（公開統計のみ）
  • チャンネル ID、チャンネル名、登録者数（公開分のみ）、アップロード再生リスト ID
取得する情報は 一般公開されている範囲 に限られ、限定公開・非公開動画やプライベートチャンネルの情報は取得しません。

2.3 取得しない情報

  • 視聴者の個人情報（メールアドレス、氏名、住所等）
  • 動画作成者・チャンネル所有者の個人情報（公開プロフィール以外）
  • 非公開のコメントや限定公開動画
  • YouTube ユーザーの認証情報、視聴履歴、検索履歴

3. 情報の利用目的

取得した情報は次の目的でのみ利用します：

  • 個人VTuberの動画・チャンネルのバズ動向を集計・ランキング表示する
  • 過去のデータと比較した時系列分析を行う
  • 傾向分析レポート（ヒントレポート）を生成する

取得情報は 第三者への販売・提供は一切行いません。広告配信、マーケティング目的のプロファイリング、レコメンドエンジン構築には利用しません。

4. データの保存期間

  • 取得した動画・チャンネル情報は、本プロジェクトの GitHub リポジトリ上の SQLite データベースおよび JSON ファイルとして無期限に保存します（履歴比較のため）
  • ただし YouTube 上で削除された動画・チャンネルについては、運営者が把握した時点で本サービス側のデータも削除します
  • 動画作成者・チャンネル所有者から削除依頼があった場合は、原則として 7 日以内にデータを削除します（連絡先：上記メール）

5. 第三者への提供

取得情報を第三者に提供することはありません。ただし以下の場合は除きます：

  • 法令に基づく開示請求があった場合
  • YouTube / Google からの正式な要請があった場合

6. YouTube API サービスの利用について

本サービスは YouTube API Services を利用しています。本サービスを利用することで、利用者は以下に同意したものとみなされます：
  • YouTube 利用規約 (YouTube Terms of Service)
  • Google プライバシー ポリシー
本サービスの YouTube 関連表示部分は、YouTube ブランドガイドラインに準拠してロゴおよびリンクを表示します。

7. データ削除リクエスト

動画作成者・チャンネル所有者から以下の連絡を受けた場合、対応いたします：

  • 自身の動画・チャンネル情報の削除依頼
  • 本サービスに保存されているデータの開示請求
  • その他プライバシーに関する問い合わせ

連絡先: amtokyo713@gmail.com（カイン宛）

1. Overview

"VTuber Buzz Ranking" (hereinafter "the Service") is an individually operated, open-source project for observing the trends of videos and posts by individual VTubers (Virtual YouTubers). The Service uses YouTube Data API v3, TikTok-related APIs, and Twitter(X)-related APIs to collect publicly available information and display it in ranking format.

2. Information We Collect

2.1 Information from Service Users

The Service is delivered as static HTML / JSON files hosted on GitHub Pages, and does not collect any personal information from users on the server side. Specifically:

  • No user registration, login, or authentication required
  • No cookies or session IDs are issued
  • No IP addresses, device information, or behavioral logs are collected
  • No access analytics tools (such as Google Analytics) are installed

2.2 Information Retrieved via External APIs

The Service retrieves only publicly available information from each platform's API:

Information retrieved from YouTube Data API v3:
  • Video ID, title, description, thumbnail URL, publish date, video length
  • Video view count, like count, comment count (public statistics only)
  • Channel ID, channel name, public subscriber count, uploads playlist ID
The retrieved data is strictly limited to publicly accessible information. We do not retrieve unlisted/private videos or private channel data.

2.3 Information We Do NOT Collect

  • Personal information of viewers (email, name, address, etc.)
  • Personal information of video creators / channel owners (beyond their public profile)
  • Private comments or unlisted videos
  • YouTube users' authentication credentials, viewing history, or search history

3. Purpose of Use

Retrieved data is used solely for the following purposes:

  • Aggregating and displaying buzz trends of individual VTubers' videos and channels
  • Performing time-series analysis by comparing with historical data
  • Generating trend analysis reports

We never sell or transfer retrieved information to third parties. The data is not used for advertising, marketing-purpose profiling, or recommendation engine training.

4. Data Retention

  • Retrieved video and channel data is stored indefinitely as SQLite databases and JSON files in the project's GitHub repository (for historical comparison purposes)
  • If a video or channel is removed from YouTube, we will remove the corresponding data from our service when we become aware of it
  • Data deletion requests from video creators / channel owners will be honored within 7 days in principle (contact: email above)

5. Disclosure to Third Parties

We do not disclose retrieved data to third parties, except in the following cases:

  • When required by law
  • When formally requested by YouTube / Google

6. About YouTube API Services

This Service uses YouTube API Services. By using this Service, users are deemed to have agreed to the following:
  • YouTube Terms of Service
  • Google Privacy Policy
YouTube-related display areas in this Service follow the YouTube branding guidelines for logos and links.

  • Requests to delete your own video or channel information
  • Disclosure requests for your data stored in the Service
  • Other privacy-related inquiries
